Этот проект представляет собой приложение, созданное на C++ с использованием библиотеки SFML. Сборка и управление проектом осуществляется с помощью CMake. Проект предназначен для платформы Windows 10.

**Требования**
- Windows 10
- C++ компилятор (например Visual Studio Code)
- CMake
- SFML 

**Установка**
1. Скачайте и установите CMake с [официального сайта](https://cmake.org/download/).
2. Установите соответствующую версию SFML, скачав её с [официального сайта SFML](https://www.sfml-dev.org/download.php).
3. Убедитесь, что пути к библиотекам SFML добавлены в системные переменные PATH.
